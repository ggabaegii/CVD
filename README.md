# CVD
동의대학교 컴퓨터소프트웨어공학과 2024 1학기 캡스톤디자인 팀프로젝트  
**색각 이상자(적록 색약)를 위한 색 보정 프로그램 (Color Correction Program for Color Vision Deviate)**
***
## 1. 프로젝트 개요
본 프로젝트는 색각 이상자, 특히 적록색약 보유자를 위한 색 보정 프로그램으로 빨강/초록 계열 색 구분이 어려운 사람들을 위해
실시간으로 찍은 사진, 갤러리에 있는 사진을 보정하여 색 인식을 돕는 기능을 구현하였습니다.
- 핵심 기능
  - 카메라 실시간 촬영 및 보정 적용
  - 갤러리 이미지 선택 후 보정 적용
  - HSV 기반 색상 강조
---
## 2. 개발 환경
- IDE : Android Studio Hedgehog
- 언어 : Java
- 라이브러리 : OpenCV 4.8.0 (Android SDK)
- 테스트 환경 : Android 14
